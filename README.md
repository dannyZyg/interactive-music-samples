# Interactive Music Samples

An interactive audio sample player for exploring music samples across different players and intensity levels.

### What Changed:
- ✅ **Complete rewrite** from Vue.js to vanilla JavaScript
- ✅ **Zero dependencies** - no npm packages, no security vulnerabilities
- ✅ **Single self-contained HTML file** - simpler, faster, more secure
- ✅ **Same functionality** - all features preserved

### Previous Issues Fixed:
2. Outdated Vue 2 framework (no longer supported)
3. Multiple npm dependency security vulnerabilities
4. Unnecessary build complexity

---

## Usage

Simply open `index.html` in any modern web browser. No build process, no dependencies, no installation required.

### Deployment

**CodeSandbox (Recommended):**
This repository is synced with CodeSandbox. Any commits to the main branch will automatically update the live demo.

**Local Development:**
```bash
# Clone the repository
git clone https://github.com/dannyZyg/interactive-music-samples.git

# Open in browser
open index.html
```

**Static Hosting:**
Upload `index.html` and the audio directories (`p1_C_H/`, `p1_C_L/`, `p2_C_H/`, `p2_C_L/`) to any static hosting service (GitHub Pages, Netlify, Vercel, etc.).
